# 歌词获取助手

歌词获取助手是一款专为音乐爱好者打造的歌词获取工具，支持网易云音乐和QQ音乐的歌曲搜索，提供歌词预览与下载功能，让你随时随地享受精准歌词服务。

## ✨ 功能特性

- 🔍 **多平台搜索**：支持网易云音乐和QQ音乐双平台歌曲搜索
- 🎵 **歌词预览**：实时预览歌词内容，支持原文和翻译显示
- 💾 **歌词下载**：支持 LRC 和 SRT 两种格式导出
- 🌐 **翻译支持**：可选择是否获取歌词翻译
- ⚙️ **灵活设置**：
  - 自定义搜索平台启用状态
  - 调整平台搜索优先级
  - 设置各平台搜索数量（1-50）
  - 选择歌词下载格式

## 🚀 快速开始

### 环境要求

- HBuilderX 3.0+

### 使用 HBuilderX 运行项目

#### 1. 安装 HBuilderX

- 官网地址：https://www.dcloud.io/hbuilderx.html

#### 2. 导入项目

**方式一：文件 -> 导入**
1. 打开 HBuilderX
2. 点击菜单栏 `文件` -> `导入` -> `从本地目录导入`
3. 选择本项目所在文件夹
4. 点击确定完成导入

**方式二：直接打开**
1. 打开 HBuilderX
2. 点击菜单栏 `文件` -> `打开目录`
3. 选择本项目所在文件夹

#### 3. 运行项目

**在浏览器中运行（H5）**
1. 在 HBuilderX 中打开项目
2. 点击菜单栏 `运行` -> `运行到浏览器` -> 选择浏览器
3. 项目将自动编译并在浏览器中打开

**在手机/模拟器中运行（App）**
1. 连接 Android 设备或启动模拟器
2. 点击菜单栏 `运行` -> `运行到手机或模拟器` -> 选择设备
3. 项目将自动编译并安装到设备上

## ⚙️ 配置说明

### 应用设置

在「设置」页面可以配置以下选项：

1. **搜索平台**：选择启用网易云音乐和/或QQ音乐
2. **搜索顺序**：点击平台可调整搜索优先级（结果显示顺序）
3. **搜索数量**：设置每个平台返回的搜索结果数量（1-50）
4. **歌词格式**：选择 LRC 或 SRT 格式
5. **歌词翻译**：开启/关闭歌词翻译功能

所有设置会自动保存到本地存储，下次使用时自动加载。

## 📝 使用说明

1. **搜索歌曲**
   - 在首页输入歌曲名称或歌手名
   - 点击「搜索」按钮
   - 系统将按照设置的平台顺序搜索歌曲

2. **预览歌词**
   - 点击搜索结果中的任意歌曲
   - 进入歌词预览页面
   - 歌词按时间轴分组显示，原文和翻译交替呈现

3. **下载歌词**
   - 在歌词预览页面点击「下载歌词」按钮
   - H5版本：文件将保存到浏览器默认下载目录
   - App版本：文件将保存到设备的下载目录
